My Research Papers

Paper Title : Introducing Dynamic Threshold Apriori-Based Approach for Question Prediction throughFrequent Pattern Mining

Journal: IEEE Xplore

Publication Date: July 2025

DOI: 10.1234/ijcse.2025.0123

This research proposes Dynamic Threshold Apriori (DTA) for predicting exam questions using frequent pattern mining. DTA adapts support thresholds based on itemset size, enhancing larger pattern retention. Applied to educational data, it outperforms standard Apriori in accuracy, offering both algorithmic and application-level novelty.

Keywords:
Data Mining Frequent Pattern Mining Apriori Algorithm Dynamic Support Threshold
